    • FASTPASSとは

    東大FASTPASSは東京大学の後期課程の学生・大学院生を中心に構成された自主ゼミ団体です。プログラミングを通じてアプリ開発・データサイエンスなどのスキルアップに取り組み、最短距離で成果につなげます。
    2018年8月からゼミ活動を継続しており、2020年春からサークルとしての本格活動を開始しています。

    活動内容(予定)と初期メンバー

    FASTPASSでは、スキルアップのための基礎を自主ゼミ形式で学びます。基礎から１つ１つ積み上げていき、実践的なアウトプットにつなげます。例えば初期メンバーは、大学院・専門課程で研究を続ける当事者であり、Pythonによるデータ分析・論文執筆・サービスローンチの経験者です。ともに学んできましょう。

    メンバー募集

    FASTPASSでは、ともに学ぶ仲間を募集しています。大学1,2年生または高校生で、東京大学駒場Iキャンパスでの活動が可能な方が対象です。サークル選択・進学選択に迷っている大学1,2年生、地方や非進学校の出身で情報格差で周りに負けたくない方、進路や受験大学に迷っている中高生を特に歓迎します。次回募集は2020年9月(第2期)ですが、見学や問い合わせはいつでも歓迎します。オンラインコンタクトフォームよりご連絡ください。
